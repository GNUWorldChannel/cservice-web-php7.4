CService People

Since Early 1995, a dedicated group of individuals have been bringing X (and back then, W) to the many thousand users of the Undernet.

Channel Service (X) help users keep their channels open, and help to keep ops in channels. The following people run this service for you.

CService Coders write the code that runs X (And Z on AUnet). You can find a copy of the code used to run X on Undernet at GitHub.

CService Admins setup Channels for the owners, and provide other services to the owner.

CService helpers hang out in #CService to assist the many thousand of users of X.
